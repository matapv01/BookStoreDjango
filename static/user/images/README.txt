Please create a default-avatar.png file in this directory with the following specifications:

1. Size: 150x150 pixels
2. Background color: #e9ecef (light gray)
3. Simple user icon in #adb5bd (darker gray)

Alternative: You can also use an open source default avatar image like:
- Gravatar's default image 
- A simple user icon from Font Awesome
- Any other free default avatar placeholder
